# waldofinder

*An answer to the question:  Where's Waldo?*

Este projeto foi desenvolvido para a disciplina Processamento de Imagens e Visão Computacional,
ministrado na Universidade Federal de São Carlos (Campus Sorocaba), pelo Prof. Dr. Siovani Cintra Felipussi.

Os arquivos **find&#95;waldo.m** e **find&#95;waldo.mlx** são os *scripts* responsáveis pela implementação do projeto.
As variáveis **scenario** e **reference** podem ser modificadas conforme desejado.
